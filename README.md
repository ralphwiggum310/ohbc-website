# OHBC Lite Website

Welcome to the official website for Our Church, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Modern, Responsive Design**: Looks great on all devices
- **Fast Performance**: Built with Next.js for optimal loading
- **Easy to Update**: Simple content structure for easy maintenance
- **No Database Required**: Static site generation for simplicity

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Build for production**
   ```bash
   npm run build
   npm start
   ```

## Project Structure

- `/src/app` - Main application pages and layout
- `/public` - Static files (images, fonts, etc.)
- `/src/components` - Reusable UI components

## Customization

- Update site metadata in `src/app/layout.tsx`
- Modify styles in `src/app/globals.css`
- Update content in the respective page files

## Deployment

This is a Next.js app that can be deployed to any static site host or Vercel for optimal performance.
